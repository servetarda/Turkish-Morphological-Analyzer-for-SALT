## 🏷️ Morfolojik Etiket Sözlüğü (Tagset)

Bu projede kullanılan etiketler, Türkçe dilbilgisi kurallarına ve SALT standartlarına uygun olarak kategorize edilmiştir.

### 1. İsim ve İsim Soylu Kelime Ekleri (Noun Suffixes)
| Etiket | Açıklama | Örnek |
| :--- | :--- | :--- |
| **PL** | Çoğul Eki | -lar, -ler |
| **1SGP** | 1. Tekil Şahıs İyelik (Benim) | -(ı)m, -(i)m |
| **2SGP** | 2. Tekil Şahıs İyelik (Senin) | -(ı)n, -(i)n |
| **3SGP** | 3. Tekil Şahıs İyelik (Onun) | -(ı), -(i), -sı, -si |
| **1PLP** | 1. Çoğul Şahıs İyelik (Bizim) | -(ı)mız, -(i)miz |
| **2PLP** | 2. Çoğul Şahıs İyelik (Sizin) | -(ı)nız, -(i)niz |
| **3PLP** | 3. Çoğul Şahıs İyelik (Onların) | -ları, -leri |
| **A** | Yaklaşma / Yönelme Durumu | -a, -e |
| **D** | Bulunma Durumu | -da, -de, -ta, -te |
| **L** | Belirtme Durumu | -ı, -i, -u, -ü |
| **AB** | Ayrılma / Uzaklaşma Durumu | -dan, -den, -tan, -ten |
| **G** | Tamlayan (Genitif) Eki | -ın, -in, -un, -ün, -nın |
| **I** | İlgi Eki / Vasita Eki | -la, -le (ile) |

### 2. Fiil Ekleri (Verb Suffixes)
| Etiket | Açıklama | Örnek |
| :--- | :--- | :--- |
| **PASS** | Edilgenlik Çatısı | -ıl, -in |
| **CAUS** | Ettirgenlik/Oldurganlık Çatısı | -dır, -t, -ir |
| **ABIL** | Yeterlilik Fiili | -abil, -ebil |
| **N** | Olumsuzluk Eki | -ma, -me |
| **AOR** | Geniş Zaman | -ar, -er, -ır, -ir |
| **PROG** | Şimdiki Zaman | -yor |
| **FUT** | Gelecek Zaman | -acak, -ecek |
| **DEV** | Öğrenilen Geçmiş Zaman (-miş) | -mış, -miş |
| **INEV** | Bilinen Geçmiş Zaman (-di) | -dı, -di |
| **COND** | Şart Kipi | -sa, -se |
| **NEC** | Gereklilik Kipi | -malı, -meli |
| **1SG** | 1. Tekil Şahıs (Şahıs Eki) | -m, -ım |
| **2SG** | 2. Tekil Şahıs (Şahıs Eki) | -n, -sın |
| **3SG** | 3. Tekil Şahıs (Şahıs Eki) | (Eksiz veya -dır) |
| **1PL** | 1. Çoğul Şahıs (Şahıs Eki) | -k, -ız, -uz |
| **2PL** | 2. Çoğul Şahıs (Şahıs Eki) | -nız, -sunuz |

### 3. Evrensel ve Diğer Ekler

## 🏷️ Morphological Tagset Dictionary

The tags used in this project are categorized in accordance with Turkish grammar rules and SALT standards.

### 1. Noun and Nominal Suffixes
| Tag | Description | Example |
| :--- | :--- | :--- |
| **PL** | Plural Suffix | -lar, -ler |
| **1SGP** | 1st Person Singular Possessive (My) | -(ı)m, -(i)m |
| **2SGP** | 2nd Person Singular Possessive (Your) | -(ı)n, -(i)n |
| **3SGP** | 3rd Person Singular Possessive (His/Her/Its) | -(ı), -(i), -sı, -si |
| **1PLP** | 1st Person Plural Possessive (Our) | -(ı)mız, -(i)miz |
| **2PLP** | 2nd Person Plural Possessive (Your) | -(ı)nız, -(i)niz |
| **3PLP** | 3rd Person Plural Possessive (Their) | -ları, -leri |
| **A** | Dative Case (Direction/Approach) | -a, -e |
| **D** | Locative Case (State of being in) | -da, -de, -ta, -te |
| **L** | Accusative Case (Definite Object) | -ı, -i, -u, -ü |
| **AB** | Ablative Case (Departure/Separation) | -dan, -den, -tan, -ten |
| **G** | Genitive Case (Possessor) | -ın, -in, -un, -ün, -nın |
| **I** | Instrumental / Relational Suffix | -la, -le (with) |

### 2. Verb Suffixes
| Tag | Description | Example |
| :--- | :--- | :--- |
| **PASS** | Passive Voice | -ıl, -in |
| **CAUS** | Causative Voice | -dır, -t, -ir |
| **ABIL** | Ability / Potential | -abil, -ebil |
| **N** | Negative Suffix | -ma, -me |
| **AOR** | Aorist (Simple Present/Habitual) | -ar, -er, -ır, -ir |
| **PROG** | Progressive (Present Continuous) | -yor |
| **FUT** | Future Tense | -acak, -ecek |
| **DEV** | Indirect/Reported Past Tense (-miş) | -mış, -miş |
| **INEV** | Direct/Definite Past Tense (-di) | -dı, -di |
| **COND** | Conditional Mood | -sa, -se |
| **NEC** | Necessitative Mood (Must/Should) | -malı, -meli |
| **1SG** | 1st Person Singular (Subject) | -m, -ım |
| **2SG** | 2nd Person Singular (Subject) | -n, -sın |
| **3SG** | 3rd Person Singular (Subject) | (Empty or -dır) |
| **1PL** | 1st Person Plural (Subject) | -k, -ız, -uz |
| **2PL** | 2nd Person Plural (Subject) | -nız, -sunuz |

### 3. Universal and Other Suffixes
| Tag | Description | Example |
| :--- | :--- | :--- |
| **COP** | Copula (Auxiliary Verb) | -dır, -dir, -ken |
| **NMLZ** | Nominalizer (Infinitive/Gerund) | -ma, -mak, -ış |
